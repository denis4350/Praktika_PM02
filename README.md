## Система «СЗР Производство»

Разработана автоматизированная система управления производством для предприятия по выпуску средств защиты растений (гербициды, инсектициды, фунгициды, протравители).

### Модули системы

- **`SZR_Production_API`** — бэкенд-сервер, обеспечивающий хранение данных, бизнес-логику и REST API для всех клиентов.
- **`SZR_TechnologistApp`** — рабочее место технолога: управление продуктами, рецептурами, технологическими картами, заказами, партиями, экструдером, отклонениями и отчётами.
- **`SZR_LaboratoryApp`** — рабочее место лаборанта: контроль качества сырья и готовой продукции, проведение испытаний, принятие решений.
- **`SZR_OperatorApp`** — рабочее место аппаратчика: мониторинг активных партий, выполнение шагов техпроцесса, просмотр телеметрии экструдера, отправка сообщений о проблемах.
